# The umoline Package
The `umoline` package provides commands `\Underline`, `\Midline` and `\Overline` for underlining, striking out, and overlining their text arguments.

## Note
This GitHub repository should only be used for reporting issues; the code is developed privately off GitHub on my local computer.

The official versions of the bundle are released on [CTAN](https://ctan.org/pkg/umoline).

## Reporting Issues
If you think that you have found an issue in the bundle, please use 
[the package issue tracker](https://github.com/xepersian/umoline/issues) 
to report it.
